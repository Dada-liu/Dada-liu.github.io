# TypeScript 核心
TypeScript 是 JavaScript 的一个超集，它添加了静态类型检查。本文将深入探讨 TypeScript 的核心概念。

TS 是一个有用的内容；
特别是它的类型检查，十分适合在现代软件开发中使用；
让我们一起开始吧；

首先，我们来了解一下 TypeScript 的基本概念。
包括类型、接口、泛型、枚举等。
还包括类型断言、类型守卫、类型别名、联合类型、交叉类型等。
交叉类型（Intersection Types）可以将多个类型合并为一个类型。
类型守卫（Type Guards）允许我们在运行时检查类型。
映射类型（Mapped Types）允许我们基于旧类型创建新类型。
枚举（Enums）为一组数值赋予更友好的名字。
