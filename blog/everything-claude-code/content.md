# Everything Claude Code

## 一、安装和上手 Claude Code

1. 安装 Claude Code
2. 进入用户级的 `.claude/` 目录：
```bash
cd ~/.claude/
```
3. 修改 Claude Code 配置文件 `CLAUDE.md`，添加最基本的配置：
eg：
```markdown
# 用中文回复我所有的问题
```
4. 开始使用 Claude Code，在使用的过程中总结 rules、commands、skills、hooks 等。



## 二、Everything Claude Code 的基本介绍
一个插件，提供了 commands、skills、hooks 等功能，帮助用户更方便地使用 Claude Code。
